Frequently Asked Questions

Everything you need to know about MCPSuperHero

What is Model Context Protocol (MCP)?

Model Context Protocol is a standardized framework for building AI applications that require context from multiple sources. MCPSuperHero helps you monitor, optimize, and scale MCP server infrastructure efficiently.

How long does setup take?

Most users have MCPSuperHero up and running within 5-10 minutes. Simply generate an API key, add it to your MCP servers, and start monitoring. We provide step-by-step setup guides for all major frameworks.

Can I monitor multiple servers with one account?

Yes! MCPSuperHero is built for unlimited server monitoring. Add as many servers as you need to your account, manage them all from a single unified dashboard, and set up different alert rules for each.

Is my data secure?

Security is our top priority. All data is encrypted in transit and at rest. We comply with industry standards including SOC 2 Type II, use secure API authentication, and never share your data with third parties.

Can I cancel my subscription anytime?

Absolutely. You can cancel your subscription at any time from your account settings. No long-term contracts, no hidden fees. Your data will be available for 30 days after cancellation.

Is there API access for integrations?

Yes. MCPSuperHero includes a comprehensive REST API for retrieving metrics, managing alerts, and integrating with your existing tools. Full API documentation is available in your dashboard.

What kind of support do you offer?

All subscribers get email support and access to our community forum. We aim to respond to all inquiries within 24 hours. Priority support is available for enterprise customers.
